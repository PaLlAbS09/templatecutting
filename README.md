# Admin Dashboard

A modern and responsive Admin Dashboard built using PHP, Bootstrap 5, and AdminLTE 4. This dashboard provides an intuitive user interface for monitoring business metrics, visualizing data, and managing administrative tasks efficiently.

## 🚀 Features

- Responsive Dashboard Layout
- AdminLTE 4 Integration
- Bootstrap 5 UI Components
- Interactive Sales Charts (ApexCharts)
- World Map Visualization (jsVectorMap)
- Direct Chat Module
- Statistics Cards
- Dark & Light Theme Support
- Sortable Dashboard Widgets
- Reusable PHP Components
- Mobile Friendly Design

## 📸 Dashboard Components

### Dashboard Cards
- New Orders
- Bounce Rate
- User Registrations
- Unique Visitors

### Data Visualization
- Sales Value Area Chart
- World Map Analytics
- Sparkline Statistics

### Communication Module
- Direct Chat Interface
- Contact List Panel
- Message Input System
